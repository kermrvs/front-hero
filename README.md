## Description

Application for heroes management.
Technologies:
- Vite;
- TypeScript;
- Vue3;
- SCSS;
- Axios;
- Pinia;
- VueRouter;

## Installation

```bash
$ yarn
```
## Running the app

First step you need to open index.ts in directory ./src/services/api/index.ts and change baseURL

```bash
# development
$ yarn dev

# build app
$ yarn build

```
